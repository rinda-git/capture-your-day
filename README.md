# capture-ur-day
RUNTEQ卒制アプリ | 英語アウトプット習慣化・英語表現の成長を蓄積できる英語ジャーナリングアプリ

---

# サービス概要

**capture-ur-day** は  英語学習者が日々の出来事や感情を英語で書き、  
AIの添削を通して **「間違いを恐れずアウトプットできる環境」** を提供する **英語アウトプットを習慣化するアプリ**です。

英語を学習している人が

- 自分の英語の成長を感じる
- 英語で自分の感情を表現できる
- アウトプットの習慣を作る
- 継続することで自信の構築、アウトプットに対する心理的ハードルを下げる

ことを目的としています。

---

# このアイデアが生まれた背景

## きっかけとなった体験

英語学習を続ける中で、次のような経験をしました。

- リスニングはできるし、単語は覚えられる
- オンライン英会話もしているが、**十分なフィードバックが得られない**
- そして **英語を話す・書く場面になると言葉が出てこない**

原因を振り返ると

- アウトプットしても、**自分の間違いを振り返る仕組みがない**
- **自分の好きなタイミングで、自分が知りたい（より自然な言い回しなど）表現を学べない**

という課題を感じていました。

また、  
- **間違えることへの恥ずかしさ**
- **アウトプットする機会が少ない**  
という心理的なブロックがありました。

語学学習では  「間違いながらアウトプットすること」が重要ですが、  
実際には
**恥ずかしさ → アウトプット回避 → 上達しない**
という状態になっていました。  
そこで**AIとの対話なら、間違いを気にせずアウトプットできるのではないか**と考え、このサービスを作ろうと思いました。  

---

## なぜこの問題が気になったのか

私はTOEIC965・英検準1級を取得していますが、実際のスピーキングやライティングは得意とは言えません。
  
原因を考えると  

- アウトプット回数が少ない
- アウトプット後の振り返りの仕組みがない
- 間違うことへの恐怖
- 失敗するときに感じる「恥ずかしさ」「自分へのがっかり感」を避ける心理

がありました。

AIの添削を通して  

- 間違いを修正できる
- 正しい表現を学べる
- 次のアウトプットに自信が持てる

という **成長サイクル** を作りたいと考えています。

---

# 解決したい課題

英語学習ではアウトプットが重要ですが、多くの学習者は

- 間違えることへの恥ずかしさ
- アウトプット機会の少なさ
- 自分の間違いを振り返る仕組みがない

という課題を抱えていると思います。

このアプリでは **英語アウトプット → AI添削 → 振り返り** のサイクルを作ることで    
**英語を使う心理的ハードルを下げること**を目的としています。

AIの添削によって自分の間違いを修正し小さな成功体験を積み重ねることで、  
英語を使うことへの自信を育て、新しいことに挑戦するきっかけになることを目指しています。

# 想定ユーザー

## ターゲット

- TOEIC600以上で英語の読み・聞きはある程度できるものの、英語で自分の考えや感情のアウトプットをする機会が少ない英語学習者
- 特に留学希望の学生や、将来ワーキングホリデー・海外就職を考えている20代～30代の学習者を想定しています。


## 利用シーン

- 通勤・退勤時間に5分、寝る前に10分　その日の感情や出来事を英文で記録します。
- AIが文法ミスや自然な言い回しで添削します。
- ユーザーは新しい表現を学び、自分の表現や間違いのパターンに気づきます。
- 添削結果の中から、ユーザーが間違えた表現が保存されます。
- 後日LINEで間違えた表現のメッセージが届きます。
- ユーザーは通勤中や寝る前に見返して、次のオンライン英会話でその表現を使ってみます。

  ※日記は日本語で書くことも可能です。

- ジャーナリングを書くことで小さな達成感を感じます。
- その日の1日や自分の感情を英語で表現でき、新しい表現や単語に出会うことで「学んだ」「新しい発見」をした小さな喜びを感じます。
- オンライン英会話や海外の友人に「この表現使ってみよう」と前向きな気持ちになれます。

# それでもこのアプリを作る理由

ChatGPTやGeminiでも英文添削は可能ですが、やりとりが単発になりやすく、  
過去の自分の誤りを継続的に振り返る導線は弱いと感じました。  

既存サービスを使う中で以下の課題を感じました。

- 添削ボタンが分かりにくい
- 元の文章が振り返りにくい
- いつ書いた文章なのかわからなくなり、学習履歴として活用しにくい
- チャットのやりとりが長くなり、過去の間違い表現にたどり着くのが大変

そのため「いつ・何を書いたのか」を簡単に振り替えられる仕組みが必要だと感じました。  
また、AIを使って英語日記を書いてノートやNotionにまとめていましたが手間になり途中で続かなくなることがありました。  

英語学習では、同じ表現や単語に何度も触れることで記憶に定着します。  
しかし、振り返る機会が少ないと、その学習サイクルが途切れてしまいます。  
そこでこのアプリは  
「日記を書く → 添削 → 間違い保存　→ アプリ内 or LINEで復習」  
という学習サイクルを1つの体験として設計しています。  

**自分が間違えた表現をLINEで自動送信する仕組み**を作り、日常で自然に復習できるようにしたいと考えています。

「LINEを開く→自分の間違った表現（正しい表現）を見る」というスモールステップを通じて  
ストレスを感じることなく英語表現に触れる機会を増やして、記憶に定着させることを促します。

このアプリでは

**「自分の間違いを蓄積して学習できる仕組み」**

を提供することを目的としています。
